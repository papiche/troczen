## Comment télécharger

### Via une passerelle IPFS publique
Cliquez simplement sur le lien ci-dessus. La passerelle `ipfs.copylaradio.com` servira le fichier.

### Via IPFS en local
Si vous avez un nœud IPFS local :
```bash
ipfs get QmVC5oxVETVRyRVaoX16dF2vA3VyXMyR7DGY3RVdW3LiiU/troczen-3.6.1.apk
```

### Via d'autres passerelles
Vous pouvez remplacer `ipfs.copylaradio.com` par d'autres passerelles :
- `ipfs.io`
- `dweb.link`
- `cloudflare-ipfs.com`

Exemple : `https://ipfs.io/ipfs/QmVC5oxVETVRyRVaoX16dF2vA3VyXMyR7DGY3RVdW3LiiU/troczen-3.6.1.apk`

## Vérification

Les APK sont signés avec la clé de signature TrocZen. Vérifiez toujours la signature avant installation.
