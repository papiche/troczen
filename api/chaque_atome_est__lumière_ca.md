# ANNEXES

## Spectres d'absorption principaux : Pain, Vin, Fromage

### Pain (amidon + gluten + croûte de Maillard)
- **Amidon (C₆H₁₀O₅)n** : absorption forte dans l'infrarouge moyen (liaisons C-O-C vers 1000-1150 cm⁻¹). Capture l'énergie thermique, stockage lent.
- **Gluten (protéines)** : bandes amide I et II (1650 et 1550 cm⁻¹). Structure β et α-hélice — géométrie φ dominante.
- **Croûte (Maillard)** : molécules aromatiques complexes, absorption UV-visible élargie. La "noirceur" au sens littéral.

### Vin (éthanol + polyphénols + anthocyanes)
- **Éthanol** : absorption IR forte (2900-3400 cm⁻¹, liaisons O-H et C-H)
- **Resvératrol, anthocyanes** : absorption UV-visible 280-520 nm. Ce sont des capteurs de lumière solaire stockée dans le raisin. Antioxydants = molécules qui "cèdent" des électrons facilement.
- **Tannins** : larges bandes phénoliques 1600 cm⁻¹

### Fromage (caséines + lipides + ferments)
- **Caséines** : structure micellaire, absorption IR similaire aux protéines mais avec phosphorylation — liaisons P-O vers 1080 cm⁻¹
- **Lipides saturés** : C-H stretching 2850-2960 cm⁻¹, stockage d'énergie dense
- **Ferments lactiques** : molécules courtes (acide lactique), spectre simple

---

### La combinaison

Ce qui est réellement intéressant : les polyphénols du vin et les protéines du fromage créent une **interaction directe** — les tannins se lient aux caséines (c'est pour ça qu'on associe fromage et vin depuis des millénaires, ça réduit l'astringence). Les spectres se *complètent* : le vin apporte la mobilité électronique (φ), le fromage apporte la structure lipidique (×2), le pain apporte le substrat énergétique lent.

C'est une combinaison cohérente spectralement.

## Triptyques Cohérents (φ)(×2)

### Principe de sélection
- **φ** : molécule mobile, polyphénolique, spiralée, donneuse d'électrons
- **×2** : structure stable, minéralisée, répétitive
- **Pont** : molécule amphiphile qui fait l'interférence entre les deux

---

### Alimentaires

**Huile d'olive / Olives / Pain au levain**
- φ : oleocanthal (polyphénol, anti-inflammatoire puissant, spectre UV 280nm)
- ×2 : acide oléique (lipide long, structure régulière)
- Pont : levain (ferments vivants, interface entre minéral et organique)

**Curcuma / Poivre noir / Ghee**
- φ : curcumine (absorption 420nm, jaune solaire)
- ×2 : beurre clarifié (lipide saturé stable)
- Pont : pipérine (augmente l'absorption de la curcumine de 2000%)

**Miel / Pollen / Eau de source**
- φ : flavonoïdes du pollen (large spectre UV-visible)
- ×2 : fructose cristallisable (structure géométrique pure)
- Pont : eau structurée (médiateur)

**Cacao cru / Sel marin / Orange**
- φ : théobromine + flavanols (spectre 270nm)
- ×2 : minéraux du sel (cristal parfait)
- Pont : vitamine C (stabilise et transporte les polyphénols)

---

### Environnementaux

**Forêt de pins / Argile / Soleil matinal**
- φ : terpènes (pinène, absorption UV, volatils spiralés)
- ×2 : argile (structure cristalline en feuillets, capacité noire géologique)
- Pont : lumière 480nm (bleu matin, synchroniseur circadien)

**Bain froid / Respiration lente / Sons graves (ton grave du funk/techno)**
- φ : hyperoxie post-apnée (flux vital)
- ×2 : contraction vasculaire (structure)
- Pont : vibration sonore infra-basse (cohérence vagale)

**Terre nue sous les pieds / Vent / Odeur de pluie (petrichor)**
- φ : géosmine (molécule du petrichor, spectre olfactif profond)
- ×2 : électrons libres du sol (earthing, mise à la terre réelle)
- Pont : ionisation de l'air par le vent
